# Bofuri-June - Site Web NightOfKarma

## Présentation

Ce projet est un site web dédié au personnage de roleplay "NightOfKarma" (June Zan) dans l'univers inspiré de l'anime "BOFURI: I Don't Want to Get Hurt, so I'll Max Out My Defense". Il s'agit d'une vitrine interactive présentant le personnage, ses caractéristiques, compétences et aventures dans le monde virtuel de "New World".

## Structure du site

Le site est composé de plusieurs pages :

- **Accueil** (index.html) : Présentation générale du personnage et du projet
- **Personnage** (personnage.html) : Biographie détaillée, histoire et relations de June/NightOfKarma
- **Monde** (monde.html) : Description de l'univers de New World et du système d'artisanat
- **Compétences** (skills.html) : Liste des compétences et talents du personnage
- **Statistiques** (statistiques.html) : Caractéristiques chiffrées et équipement
- **Galerie** (galerie.html) : Collection d'images et illustrations

## Caractéristiques du personnage

**__Nom :__** Zan

**__Prénom :__** June

**__Âge :__** 19 ans

**__Sexe :__** Fille

**__Caractère :__** Calculatrice et gentille, June analyse toujours la situation avant d'agir.
Elle est réfléchie, stratège, mais n'hésite pas à tendre la main à ceux qui en ont besoin. Sa curiosité et sa
débrouillardise l'ont menée plus loin qu'elle ne l'aurait imaginé.

**__Pseudo ingame:__** NightOfKarma

**__Classe:__** Artisan (spécialisée dans la création d'objets)

**__Stats (100 points répartis):__**
- **STR (Force):** 25
- **AGI (Agilité):** 0
- **INT (Intelligence):** 20
- **VIT (Vitalité):** 5
- **DEX (Dextérité):** 45
- **PV:** 10
- **MP:** 5

## Comment visualiser le site

1. Clonez ce dépôt sur votre ordinateur
2. Ouvrez l'un des fichiers HTML dans votre navigateur web
3. Alternativement, vous pouvez héberger ces fichiers sur un serveur web

## Crédits

Ce site est un projet de roleplay fictif inspiré par l'univers de BOFURI. Il a été conçu pour mettre en valeur le personnage de June Zan / NightOfKarma, une artisane talentueuse spécialisée dans la création d'objets uniques.
